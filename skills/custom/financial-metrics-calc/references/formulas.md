# 财务指标公式大全

## 盈利能力

### ROE（净资产收益率）
```
ROE = 净利润 / 平均股东权益 × 100%
```
- **平均股东权益** = (期初股东权益 + 期末股东权益) / 2
- **简化版**（使用期末值即可，误差通常 < 1ppt）
- 🟢 优秀 > 20%，🟡 一般 10%–20%，🔴 低 < 10%

### 毛利率
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```
- 行业差异极大，必须和同业对比
- 白酒: 60%–90% | 制造业: 15%–35% | 零售: 10%–30%

### 净利率
```
净利率 = 净利润 / 营业收入 × 100%
```
- 用**扣非净利润**更准确（排除一次性收益）

### 营业利润率
```
营业利润率 = 营业利润 / 营业收入 × 100%
```
- 比净利率更"纯粹"——剔除了营业外收支和所得税的影响

### EBITDA 率
```
EBITDA率 = EBITDA / 营业收入 × 100%
```
- 排除了折旧摊销的"非现金"影响，适合重资产行业对比

## 杜邦分析

### 三因子模型
```
ROE = 净利率 × 资产周转率 × 权益乘数

净利率 = 净利润 / 营业收入
资产周转率 = 营业收入 / 总资产
权益乘数 = 总资产 / 股东权益
```

### 五因子模型
```
ROE = (净利润/EBT) × (EBT/EBIT) × (EBIT/营收) × (营收/总资产) × (总资产/股东权益)
      税负因子      利息负担     经营利润率    资产周转率      权益乘数
```

## 偿债能力

### 流动比率
```
流动比率 = 流动资产 / 流动负债
```
🟢 > 2.0  |  🟡 1.0–2.0  |  🔴 < 1.0

### 速动比率
```
速动比率 = (流动资产 - 存货) / 流动负债
```
🟢 > 1.0  |  🟡 0.5–1.0  |  🔴 < 0.5

### 资产负债率
```
资产负债率 = 总负债 / 总资产 × 100%
```
🟢 < 40%  |  🟡 40%–60%  |  🔴 > 70%（非金融行业）

### 利息覆盖倍数
```
利息覆盖倍数 = EBIT / 利息费用
```
🟢 > 5  |  🟡 2–5  |  🔴 < 2

### 有息负债率
```
有息负债率 = (短期借款 + 长期借款 + 应付债券 + 一年内到期非流动负债) / 股东权益 × 100%
```
🔴 > 100%（有息负债超过净资产）

## 现金流质量

### OCF/NI 比
```
OCF/NI = 经营活动现金流量净额 / 净利润
```
🟢 > 1.0  |  🟡 0.5–1.0  |  🔴 < 0.5

### 自由现金流
```
FCF = 经营活动现金流量净额 - 资本支出
```
- 资本支出 = 购建固定资产、无形资产和其他长期资产支付的现金

### FCF Yield
```
FCF Yield = FCF / 市值 × 100%
```
🟢 > 8%  |  🟡 3%–8%  |  🔴 < 3%

## 成长性

### 同比增长率（YoY）
```
YoY = (本期 - 去年同期) / |去年同期| × 100%
```

### 复合增长率（CAGR）
```
CAGR = (期末值 / 期初值)^(1/n) - 1
```
n = 年数

### PEG（市盈率相对增长）
```
PEG = PE / 盈利增速(%)
```
🟢 < 1  |  🟡 1–2  |  🔴 > 2

## 营运效率

### 存货周转天数
```
存货周转天数 = 365 / (营业成本 / 平均存货)
```
越小越好（卖得快）

### 应收账款周转天数
```
应收周转天数 = 365 / (营业收入 / 平均应收账款)
```
越小越好（回款快）

### 总资产周转率
```
总资产周转率 = 营业收入 / 平均总资产
```
- 轻资产公司 > 1.0 正常，重资产公司 0.3–0.8 正常

## 估值指标

### PE（市盈率）
```
PE = 股价 / 每股收益 = 市值 / 净利润
```
- 适合盈利稳定的公司
- 周期股在盈利高点 PE 低 → 可能是卖出信号

### PB（市净率）
```
PB = 股价 / 每股净资产 = 市值 / 净资产
```
- 适合金融、重资产行业
- PB < 1 不一定是"便宜"，可能是资产质量差

### PS（市销率）
```
PS = 市值 / 营业收入
```
- 适合高增长但尚未盈利的公司

### EV/EBITDA
```
EV = 市值 + 总负债 - 货币资金
EV/EBITDA = EV / EBITDA
```
- 排除了资本结构和折旧政策的影响
- 更适合跨公司、跨市场对比

## 综合预警

### Altman Z-score
```
Z = 1.2×(营运资金/总资产)
  + 1.4×(留存收益/总资产)
  + 3.3×(EBIT/总资产)
  + 0.6×(市值/总负债)
  + 1.0×(营收/总资产)
```
🟢 > 2.99  |  🟡 1.81–2.99  |  🔴 < 1.81
