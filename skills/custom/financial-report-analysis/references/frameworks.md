# 财报分析框架详解

## 1. 杜邦分析体系

### 三因子模型
ROE = 净利率 × 资产周转率 × 权益乘数

### 五因子模型（进阶，需要更细的数据）
ROE = 税负因子 × 利息负担因子 × 经营利润率 × 资产周转率 × 权益乘数

### 解读方法
- 逐年对比各因子的变化，锁定 ROE 变动的主导因素
- 和同行业公司对比，看差异来自哪个因子
- 警惕高杠杆拉动的 ROE（权益乘数 > 3 时需关注风险）
- 净利率和周转率的涨跌方向相反时，分析公司是否在"以价换量"或"以量换价"

## 2. 现金流分析框架

### 三步分析法

**第一步：判断现金流画像**

| 类型 | 经营CF | 投资CF | 筹资CF | 特征 |
|------|--------|--------|--------|------|
| 现金牛 | + | - | - | 成熟期优质公司 |
| 成长期 | + | - | + | 高成长公司 |
| 衰退期 | - | + | - | 变卖资产维持 |
| 危险期 | - | - | + | 经营不赚钱，靠融资续命 |

**第二步：检查 OCF/NI 比**
- 正常 > 0.8，优秀 > 1.2
- 如果常年 < 0.5，利润质量堪忧

**第三步：计算自由现金流**
- FCF = 经营现金流 - 资本支出
- 连续 FCF > 0 是分红和回购的资金来源
- 如果 FCF 为负但营收高速增长，可能是"战略性亏损"（不一定坏事）

## 3. 成长性分析框架

### CAGR（复合年增长率）
CAGR = (终值/初值)^(1/n) - 1

### 增长质量三层次

| 层次 | 指标对比 | 含义 |
|------|---------|------|
| 营收增长 | 与行业均值比 | 是否在抢占份额 |
| 利润增长 vs 营收增长 | 利润增速 > 营收增速 | 有规模效应 |
| 现金流增长 vs 利润增长 | 现金流增速 > 利润增速 | 真正的现金增长 |

**警惕信号：**
- 营收增长 + 经营现金流恶化 → 增长的"水分"（应收账款堆积）
- 利润增长 + 存货暴增 → 可能滞销

## 4. 同业对比框架

### 步骤
1. 选择 3-5 家可比公司（同行业、相近市值）
2. 取同样的时间窗口
3. 比较核心指标：ROE、毛利率、净利率、负债率、营收增速
4. 标注异常值和需要解释的部分

### 多维度排名法

| 维度 | 权重 | 公司A排名 | 公司B排名 |
|------|------|----------|----------|
| ROE | 25% | | |
| 毛利率 | 20% | | |
| 营收增速 | 20% | | |
| OCF/NI比 | 20% | | |
| 资产负债率 | 15% | | |
| **加权总分** | 100% | | |

## 5. 估值分析框架

### PE 估值
- 适合盈利稳定的公司
- PE = 市值 / 净利润
- 历史分位法比绝对值更有意义

### PEG（市盈率相对增长）
- PEG = PE / 盈利增长率(%)
- < 1 → 可能低估
- > 2 → 估值已反映高增长预期

### FCF Yield
- FCF Yield = 自由现金流 / 市值
- > 5% → 有吸引力
- 比 PE 更"诚实"（现金流比利润难操纵）

### EV/EBITDA
- 排除资本结构和折旧政策的影响
- 更适合跨公司对比
- 适合重资产行业
