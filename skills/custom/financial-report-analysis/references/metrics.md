# 财务指标参考手册

## 盈利能力指标

### ROE（净资产收益率）
- **公式：** 净利润 / 平均股东权益 × 100%
- **含义：** 衡量股东投入资本的回报效率。巴菲特最看重的指标之一。
- **合理范围：**
  - 一般制造业：8%–15%
  - 消费品牌公司：15%–30%
  - 金融行业：10%–18%
  - 白酒行业：15%–35%
- **判断标准：** 连续 5 年以上 ROE > 15% 是高质量公司的基本特征
- **杜邦拆解：** ROE = 净利率 × 资产周转率 × 权益乘数
  - 净利率高 → 品牌/技术/定价权优势
  - 周转率高 → 运营效率优秀（薄利多销型）
  - 杠杆高 → 依赖债务驱动（需关注风险）

### 毛利率（Gross Margin）
- **公式：** (营业收入 - 营业成本) / 营业收入 × 100%
- **合理范围（因行业差异巨大）：**
  - 制造业：15%–35%
  - 消费品品牌：40%–70%
  - 白酒行业：60%–90%
  - SaaS/软件：70%–90%
  - 零售业：10%–30%
- **解读要点：**
  - 毛利率持续下降 → 竞争加剧或成本上升（可能是被动降价）
  - 毛利率显著高于同行 → 品牌溢价或技术壁垒
  - 毛利率提升但市占率下降 → 可能正在"高端化"但失去大众市场

### 净利率（Net Margin）
- **公式：** 净利润 / 营业收入 × 100%
- **合理范围：** 因行业差异大。制造业 5%–15%，消费品牌 10%–25%，互联网平台 20%–40%
- **注意：** 需区分"净利润"和"扣非净利润"。如果非经常性损益占比过高（>30%），扣非净利率更有参考价值

## 偿债能力指标

### 资产负债率
- **公式：** 总负债 / 总资产 × 100%
- **合理范围：**
  - 制造业：40%–60%
  - 重资产行业（电力、基建）：50%–70%
  - 房地产：60%–80%（行业特性，高负债正常）
  - 轻资产/品牌公司：20%–40%
- **解读：** 并非越低越好。适度杠杆可以提升 ROE，但过高增加破产风险

### 流动比率
- **公式：** 流动资产 / 流动负债
- **合理范围：** 1.5–2.5
- **危险信号：** < 1.0（短期偿债能力不足）

### 速动比率
- **公式：** (流动资产 - 存货) / 流动负债
- **合理范围：** > 1.0
- **危险信号：** < 0.5

### 利息覆盖倍数
- **公式：** EBIT / 利息费用
- **合理范围：** > 3（安全），< 2（关注），< 1（危险）

## 现金流质量指标

### 经营现金流/净利润比（OCF/NI）
- **公式：** 经营活动现金流量净额 / 净利润
- **合理范围：**
  - > 1.0：利润质量高，现金回收好
  - 0.5–1.0：一般，需关注
  - < 0.5：利润可能"虚"，需重点关注应收账款和存货
  - 长期 < 0：经营不可持续

### 自由现金流（FCF）
- **公式：** 经营现金流 - 资本支出
- **FCF Yield：** FCF / 市值
- **判断：** 持续正 FCF 是"现金牛"的核心特征

## 成长性指标

### 同比增长率（YoY Growth）
- **公式：** (本期 - 去年同期) / |去年同期| × 100%
- **关注点：** 营收增速 vs 利润增速 vs 经营现金流增速
  - 利润增速 > 营收增速 → 规模效应或成本优化
  - 经营现金流增速 > 利润增速 → 增长质量高

### 复合年增长率（CAGR）
- **公式：** CAGR = (终值/初值)^(1/n) - 1
- **适用：** 评估中长期（3-5年）的成长趋势

## 综合预警指标

### Altman Z-score
- **公式：** Z = 1.2X1 + 1.4X2 + 3.3X3 + 0.6X4 + 1.0X5
- **判定：** > 2.99 安全，1.81–2.99 灰色区，< 1.81 危险
- **局限：** 原模型针对美国制造业，对金融/科技行业适用性有限
