# 财务风险评估

## Overview

此技能系统性地审查上市公司的财务风险，覆盖六大风险维度。核心原则：**风险分析要找"不对劲"的地方——不符合商业逻辑的科目变动、与同行显著偏离的指标、难以解释的现金流异常。**

## 风险评估工作流

### 维度 1: 偿债能力风险

#### 短期偿债

| 指标 | 公式 | 危险信号 |
|------|------|---------|
| 流动比率 | 流动资产 / 流动负债 | < 1.0（流动资产不足以覆盖短期债务） |
| 速动比率 | (流动资产 - 存货) / 流动负债 | < 0.5（扣除存货后偿债能力极弱） |
| 现金比率 | 货币资金 / 流动负债 | 持续下降 + 经营现金流恶化 |

#### 长期偿债

| 指标 | 公式 | 危险信号 |
|------|------|---------|
| 资产负债率 | 总负债 / 总资产 | > 70%（非金融行业） |
| 利息覆盖倍数 | EBIT / 利息费用 | < 2（利润不够付利息） |
| 有息负债率 | 有息负债 / 股东权益 | > 100%（借的钱超过自有资金） |

**额外检查：**
- 短期借款 + 一年内到期非流动负债 是否远超货币资金？
- 是否存在"存贷双高"？（货币资金多 + 有息负债多 → 造假红旗）

### 维度 2: 现金流质量

这是最重要的风险维度——**利润可以粉饰，现金流很难造假。**

#### 核心指标

| 指标 | 公式 | 危险信号 |
|------|------|---------|
| OCF/NI 比 | 经营现金流 / 净利润 | 长期 < 0.5（利润没有现金支撑） |
| FCF | 经营现金流 - 资本支出 | 连续多年为负（烧钱模式不可持续） |
| FCF/NI 比 | 自由现金流 / 净利润 | 长期 < 0（赚了利润但留不下现金） |

#### 现金流画像（三种典型模式）

| 类型 | 经营CF | 投资CF | 筹资CF | 解读 |
|------|--------|--------|--------|------|
| 🟢 现金牛 | + | - | - | 经营赚钱→投资扩张+还债/分红，健康 |
| 🟡 成长期 | + | - | + | 经营赚钱但不够花，靠融资扩张 |
| 🔴 危险 | - | - | + | 经营不赚钱，靠借钱/融资续命 |
| 🔴 更危险 | + | + | + | 经营赚钱但还在变卖资产+融资？（异常） |

### 维度 3: 资产质量风险

#### 应收账款质量

- 应收账款增速 > 营收增速 → 可能放宽信用政策冲收入
- 应收账款周转天数持续增长 → 回款变慢
- 大额单项计提坏账 → 客户质量恶化

#### 存货质量

- 存货增速 > 营收增速 → 可能滞销
- 存货周转天数持续增长 → 去化困难
- 大额存货跌价准备 → 产品贬值

#### 商誉风险

- 商誉 / 净资产 > 30% → 减值风险敞口大
- 被收购公司业绩不达预期 → 减值测试可能触发
- 上市公司从未计提过商誉减值 → 需要关注

### 维度 4: Altman Z-score（综合预警）

**原始 Z-score（制造业）：**
```
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5

X1 = 营运资金 / 总资产
X2 = 留存收益 / 总资产
X3 = EBIT / 总资产
X4 = 市值 / 总负债
X5 = 营收 / 总资产
```

| Z-score | 判定 |
|---------|------|
| > 2.99 | 🟢 安全区 |
| 1.81 – 2.99 | 🟡 灰色区（需关注） |
| < 1.81 | 🔴 危险区（破产风险高） |

### 维度 5: 利润质量检验

| 检查项 | 方法 | 红旗 |
|--------|------|------|
| 非经常性损益占比 | 扣非净利润 / 净利润 | < 0.7（利润靠一次性收益撑） |
| 毛利率异常变动 | 同比/环比变化 > 5ppt | 无合理解释的剧烈波动 |
| 费用率异常 | 销售/管理/研发费率同比变化 | 收入下降但费用刚性不减 |
| 关联交易占比 | 关联交易 / 营收 | > 30%（利润可能被操纵） |
| 审计意见 | 查看年报审计意见 | 非标准无保留意见（保留/否定/无法表示） |

### 维度 6: 公司治理风险（辅助）

通过 `web_search` 补充：
- 大股东股权质押比例（> 50% 需关注）
- 近期是否有监管问询/立案调查
- 高管频繁变更/财务总监离职
- 分红政策是否稳定

## 风险评级

综合六大维度，给出风险评级：

| 评级 | 说明 |
|------|------|
| 🟢 低风险 | 各项指标正常，无重大风险信号 |
| 🟡 关注 | 1-2 项指标偏离正常范围，建议持续跟踪 |
| 🟠 警惕 | 多个维度出现异常信号，需要深入调查 |
| 🔴 高风险 | 存在财务造假或破产风险的典型信号 |

## 输出格式

```
## 🔍 {公司名称} 财务风险评估

**综合评级：** 🟡 关注

### 偿债能力
- 流动比率：{值}（{解读}）
- 资产负债率：{值}（{解读}）

### 现金流质量
- OCF/NI比：{值}（{解读}）
- 现金流画像：🟢 现金牛型

### 资产质量
- 应收账款周转天数：{值}（{解读}）
- 商誉/净资产：{值}（{解读}）

### Z-score：{值} → {判定}

### 风险总结
1. {关键风险1}
2. {关键风险2}

### 建议关注
- {需要持续跟踪的指标}
```

> ⚠️ **重要：** 风险评估是概率判断，不是确定性预测。所有结论应标注依据的数据来源和数据截止日期，不确定的地方明确说明。
