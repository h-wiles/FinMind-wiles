# 财务风险评估模型与方法

## 1. Altman Z-score 详解

### 原始模型（制造业）
```
Z = 1.2X1 + 1.4X2 + 3.3X3 + 0.6X4 + 1.0X5
```

| 变量 | 含义 | 公式 |
|------|------|------|
| X1 | 营运资金/总资产 | 流动性 |
| X2 | 留存收益/总资产 | 累计盈利能力 |
| X3 | EBIT/总资产 | 资产盈利能力 |
| X4 | 市值/总负债 | 市场信心 |
| X5 | 营收/总资产 | 资产周转效率 |

### 判定标准
- 🟢 Z > 2.99：安全区 — 破产概率低
- 🟡 1.81 ≤ Z ≤ 2.99：灰色区 — 需要持续关注
- 🔴 Z < 1.81：危险区 — 破产风险较高

### 适用性限制
- 原始模型针对**美国制造业**上市公司
- 对金融、科技、服务业适用性较弱
- 中国 A 股市场修正观点：Z 值整体下移 0.5–1.0（因为 PE 普遍更高导致 X4 偏高。但偏保守地，不建议自行调整阈值，Z-score 低就是低）
- 不应作为唯一依据，需结合其他维度综合判断

## 2. 现金流质量检验

### 方法：三表交叉验证

**检查 1：利润是否有现金支撑？**
```
OCF/NI 比 = 经营现金流 / 净利润
```
- 理想 > 1.0
- 如果常年 < 0.5：净利润有"水分"的可能

**检查 2：营收增长的同时应收账款是否同步增长？**
```
应收账款增速 / 营收增速
```
- > 1.5 → 危险（大量销售挂账，未收到现金）
- < 0.8 → 健康

**检查 3：存货增长是否合理？**
```
存货增速 / 营收增速
```
- > 1.5 → 可能滞销
- < 0.8 → 健康

**检查 4：现金流画像判读**
```
经营CF  投资CF  筹资CF  解读
+       -       -      🟢 现金牛
+       -       +      🟡 成长期
-       -       +      🔴 烧钱期
-       +       +      🔴 卖资产+借钱（非常危险）
+       +       +      🔴 经营赚钱却在卖资产+融资（逻辑不通）
```

## 3. 资产负债表红旗标志

### 存贷双高
- **定义：** 货币资金很多 + 有息负债也很多
- **正常解释：** 可能有境外资金无法调回、或为了维持银行关系
- **造假红旗：** 康得新、康美药业案 → 货币资金可能是假的（虚增存款来掩盖资金占用）

### 应收账款异常
- 应收账款/营收 > 50%（制造业）
- 应收账款周转天数逐年拉长
- 大额应收账款集中在少数客户（客户集中度风险）
- 长期挂账未收回的应收款（> 1年）

### 商誉高风险
- 商誉/净资产 > 30%
- 被收购公司业绩承诺期满当年
- 连续多年未做商誉减值测试

### 其他红旗
- 大额其他应收款（可能是股东占款）
- 在建工程长期挂账不转固（延迟折旧，虚增利润）
- 研发支出资本化比例异常高（"隐藏"费用）
- 关联交易占营收比例过高（利润可操纵）

## 4. 利润表红旗标志

### 非经常性损益占比过高
```
扣非净利润 / 净利润 < 0.7
```
→ 利润主要靠卖资产、政府补贴、投资理财收益等一次性项目撑

### 毛利率异常波动
- 无合理解释的毛利率同比变化 > 5ppt
- 和同行毛利率变化方向相反

### 费用率异常
- 营收下降但费用刚性不减（可能是"洗大澡"的前兆）
- 研发费用突然大幅下降（可能是为了保利润砍研发，长期不利）

## 5. 审计信号

### 审计意见类型（从好到差）
1. 🟢 标准无保留意见 — 正常
2. 🟡 带强调事项段的无保留意见 — 有值得关注的事项
3. 🟠 保留意见 — 有部分问题
4. 🔴 否定意见 — 严重问题
5. 🔴 无法表示意见 — 审计师拒绝发表意见

- 连续多年被出具**非标准无保留意见**是大红旗
- 审计师频繁更换也是红旗

### 关键审计事项
- 阅读年报"关键审计事项"段，了解审计师认为风险最大的领域
- 如果和你的分析判断一致 → 确认风险
- 如果审计师没提但你发现异常 → 可能是审计遗漏

---

## 6. 公司治理红旗

- 大股东股权质押比例 > 50%
- 高管/财务总监频繁变更
- 近期收到交易所问询函/证监会立案调查
- 实际控制人存在大额对外担保
- 关联方资金占用
- 分红政策异常（突然不分红/突然高分红）
