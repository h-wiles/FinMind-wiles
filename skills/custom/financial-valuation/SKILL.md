# 估值分析

## Overview

此技能提供上市公司的多维度估值分析。核心原则：**估值不是一个绝对值，必须在历史和同业两个参照系下才有意义。** PE=30 对成长股可能合理，对银行股可能就贵了。

## 估值分析工作流

### Step 1: 获取估值数据

调用 `stock_info` 获取当前估值指标：
```
stock_info("600519")
→ {pe_ratio, pb_ratio, ps_ratio, market_cap, dividend_yield, 52w_high, 52w_low, ...}
```

### Step 2: 多维估值框架

#### 维度 1: 当前估值水平

| 指标 | 含义 | 适用场景 |
|------|------|---------|
| PE（市盈率） | 市值 / 净利润 | 盈利稳定的公司 |
| PB（市净率） | 市值 / 净资产 | 金融、重资产行业 |
| PS（市销率） | 市值 / 营收 | 高增长但未盈利的公司 |
| PCF（市现率） | 市值 / 经营现金流 | 现金流比利润更可靠时 |
| FCF Yield | 自由现金流 / 市值 | 衡量股东现金回报率 |
| 股息率 | 每股分红 / 股价 | 红利投资者关注 |

#### 维度 2: 历史分位（纵向对比）

判断当前估值在自身历史中处于什么位置：

| 分位 | 判定 |
|------|------|
| < 20% | 🟢 历史低位（可能低估） |
| 20% – 40% | 🟢 偏低 |
| 40% – 60% | 🟡 中等 |
| 60% – 80% | 🟠 偏贵 |
| > 80% | 🔴 历史高位（可能高估） |

**重要：**
- 至少需要 3-5 年的历史估值数据才有参考意义
- 如果公司发生过重大业务变化（并购、剥离），历史估值可能不再可比
- 新股/次新股的历史分位参考价值有限

#### 维度 3: 同业对比（横向对比）

和同行业可比公司对比估值。对比维度：
- PE / PB / PS 绝对值和分位数
- PEG（PE / 盈利增速）→ 衡量成长性价比
- FCF Yield → 衡量现金回报

**解读原则：**
- 如果业绩增速显著高于同行，享受估值溢价是合理的
- 如果增速和同行差不多但估值高出很多，需要警惕
- 周期股在盈利高点 PE 看似很低（高利润低PE），但实际上可能是卖出信号

#### 维度 4: FCF Yield 评估

FCF Yield 是判断公司"真实便宜度"的核心指标（比 PE 更难操纵）。

| FCF Yield | 判定 |
|-----------|------|
| > 8% | 🟢 极具吸引力（可能低估） |
| 5% – 8% | 🟢 较有吸引力 |
| 3% – 5% | 🟡 合理 |
| 1% – 3% | 🟠 偏贵 |
| < 1% | 🔴 昂贵（除非高增长可解释） |

**FCF Yield = (经营现金流 - 资本支出) / 市值**

#### 维度 5: 跨市场估值对比（中港美）

如果对比跨市场公司，注意以下差异：

| 因素 | A股 | 港股 | 美股 |
|------|-----|------|------|
| 流动性溢价 | 较高（散户多、换手率高） | 较低（机构为主） | 中等 |
| 行业偏好 | 消费、医药享受高估值 | 金融地产低估值 | 科技享受高估值 |
| 税收 | 股息红利税差异化 | 印花税+股息税 | 资本利得税 |
| 无风险利率 | ~2.5%（10年国债） | ~3.5% | ~4% |

**结论：** 跨市场对比时，A股天然比港股有一定估值溢价（通常 20-40%），这是流动性差异导致的，不一定意味着 A 股"贵"。

### Step 3: 综合估值判断

结合以上维度，给出综合估值判断：

| 评级 | 条件 |
|------|------|
| 🟢 明显低估 | 历史低位 + 同业偏低 + FCF Yield > 5% |
| 🟢 偏低 | 2个以上维度偏低 |
| 🟡 合理 | 各维度在合理范围 |
| 🟠 偏贵 | 2个以上维度偏高 |
| 🔴 明显高估 | 历史高位 + 同业偏高 + FCF Yield < 2% |

## 估值陷阱（必须检查）

1. **PE 低并不意味着便宜**
   - 可能是一次性收益推高利润（扣非净利润远低于净利润）
   - 可能是周期高点（盈利即将下滑）
   - 可能是资产重估带来的账面利润

2. **高增长不一定支撑高估值**
   - 增长质量比速度重要（有现金流支撑的增长 vs 靠应收堆出的增长）
   - 如果 PEG > 2，即使高增长也可能已经透支

3. **PB < 1 不一定是"破净便宜"**
   - 可能是资产有大量水分（商誉、坏账未计提）
   - 可能是 ROE 持续走低（资产不赚钱）
   - 银行/地产破净是常态，不代表低估

## 输出格式

```
## 💰 {公司名称} 估值分析

**当前估值：**
- PE：{值}（{历史分位}分位）
- PB：{值}（{历史分位}分位）
- FCF Yield：{值}%

**历史对比：**
- 过去5年 PE 范围：{min} – {max}，中位数：{median}
- 当前 PE 处于历史 {分位} 分位

**同业对比：**

| 公司 | PE | 营收增速 | PEG | FCF Yield |
|------|-----|---------|-----|-----------|
| {公司A} | | | | |
| {公司B} | | | | |
| {行业均值} | | | | |

**综合估值判断：** 🟡 合理

**风险提示：**
- {估值相关的风险因素}
```

> ⚠️ **重要：** 估值分析是对当前市场价格是否"合理"的判断，不代表未来涨跌预测。市场可能长期处于高估或低估状态。所有分析必须标注数据截止日期。
