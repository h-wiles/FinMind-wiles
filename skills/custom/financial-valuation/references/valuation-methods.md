# 估值方法论

## 相对估值法（市场法）

### PE（市盈率）
- **公式：** PE = 市值 / 净利润 = 股价 / 每股收益
- **适用：** 盈利稳定的成熟公司
- **不适用：** 亏损公司、利润波动极大的周期股
- **陷阱：** 低 PE 不等于便宜
  - 排查是否有大额非经常性收益
  - 排查是否处于盈利周期高点
  - 排查是否有资产重估带来的账面利润

### PB（市净率）
- **公式：** PB = 市值 / 净资产 = 股价 / 每股净资产
- **适用：** 金融（银行/保险/券商）、重资产行业
- **不适用：** 轻资产公司（PB 极高是正常的）
- **陷阱：** PB < 1 未必便宜
  - 如果公司 ROE < 资金成本，折价交易是合理的
  - 如果是资产质量差（商誉多、坏账多），账面净资产虚高

### PS（市销率）
- **公式：** PS = 市值 / 营业收入
- **适用：** 高速增长但尚未盈利的公司
- **陷阱：** 营收规模大但不赚钱（低利润率），PS 低不代表便宜

### PEG（市盈率相对增长）
- **公式：** PEG = PE / 盈利增长率(%)
- **判定：** < 1 可能低估，> 2 高增长已充分定价
- **陷阱：**
  - 用哪一年的增长率？过去？未来预测？后者充满不确定性
  - 增长率如果 < 5%，PEG 会严重失真

## EV/EBITDA 法

### 为什么用 EV 而不是市值？
- EV（企业价值）= 市值 + 总负债 - 货币资金
- 市值只反映股权价值，EV 反映整个企业的价值（股权+债权）
- EBITDA 排除了折旧政策、税率的差异
- **EV/EBITDA 是最适合跨公司、跨市场对比的估值指标**

### 判定参考
| 行业 | EV/EBITDA 合理范围 |
|------|-------------------|
| 消费品 | 10–20x |
| 工业制造 | 6–12x |
| 科技 | 12–25x |
| 公用事业 | 8–15x |

## FCF Yield 法

### 为什么 FCF Yield 比 PE 更"诚实"？
- PE 使用的是净利润（会计数字，可操作空间大）
- FCF 是真正的现金（经营现金流 - 资本支出），更难操纵
- FCF Yield 直观表达了"如果我买下整家公司，每年能拿回多少现金"

### 判定
| FCF Yield | 含义 |
|-----------|------|
| > 10% | 🟢 极有吸引力（但检查是否一次性因素） |
| 5%–10% | 🟢 较好 |
| 3%–5% | 🟡 合理 |
| < 3% | 🟠 偏贵 |
| 负值 | 🔴 不产生自由现金流 |

## 历史分位法

### 方法
1. 收集公司过去 3-5 年的 PE/PB 数据（月度或季度频率）
2. 剔除极端异常值（PE > 200 或 PE < 0）
3. 计算当前值在历史分布中的分位数

### 判定
| 分位 | 判定 |
|------|------|
| < 20% | 🟢 历史低位 |
| 20%–40% | 🟢 偏低 |
| 40%–60% | 🟡 中等 |
| 60%–80% | 🟠 偏贵 |
| > 80% | 🔴 历史高位 |

### 注意
- **过去不一定能预测未来。** 如果公司基本面发生了结构性变化（如业务转型、行业颠覆），历史估值区间可能不再适用
- 熊市中 PE 普遍压缩，牛市 PE 普遍扩张——分位法在极端市场中会产生系统性偏差
- 新股/次新股（<3 年上市）的历史分位参考价值有限

## 跨市场估值对比注意事项

### A股 vs 港股 vs 美股

| 因素 | A股 | 港股 | 美股 |
|------|-----|------|------|
| 散户占比 | 高（~60%） | 低（~15%） | 中等（~25%） |
| 流动性溢价 | 较高 | 较低 | 中等 |
| 无风险利率 | ~2.5% | ~3.5% | ~4.0% |
| 行业偏好 | 消费、医药高估值 | 金融地产低估值 | 科技高估值 |
| 分红税 | 差异化税率 | 10–20% | 0–30% |

### 经验法则
- A 股相对港股通常有 20%–40% 的估值溢价（流动性+散户溢价）
- 这**不一定**意味着 A 股"贵"了——两个市场的投资者结构和流动性不同
- 如果做 AH 股折溢价分析，要关注汇率波动的影响

## 估值陷阱清单

在做估值判断之前，过一遍这个清单：

1. ☐ PE < 5 但扣非净利润远低于净利润？
2. ☐ 高 ROE 是否依赖高杠杆？（权益乘数 > 3）
3. ☐ 营收增长的同时经营现金流是否恶化？
4. ☐ 是否存在大额商誉？（商誉/净资产 > 30%）
5. ☐ 是否有大量表外负债（经营租赁、担保等）？
6. ☐ 公司是否处于盈利周期高点（而市场按高点利润定价）？
7. ☐ 大股东是否在大规模减持？

任何一项为"是"，估值应该打折。

---

> **核心原则：估值是一门艺术，不是精确科学。** 它的价值在于让你大致知道自己是"花 1 块钱买了价值 2 块的东西"还是"花 5 块钱买了价值 1 块的东西"。PE 是 30 还是 28 不重要，重要的问题是："这个行业、这个增速、这个ROE，值不值得现在这个价格？"
